# TeamMatch Prototype

A React/Vite prototype for a student team project matching platform.

## Main flow

- Login
- Student profile editing
- Team recruitment post list
- Post detail view
- Join request sending
- Request accept/reject management

## Tech stack

- React
- Vite
- Framer Motion
- Lucide React
- CSS

## Run locally

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal.

## Project status

This is a frontend prototype using mock data. It is intended to demonstrate the MVP user flow for the SDLC capstone implementation stage.
